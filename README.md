# VK Bot: Optica Express (v1.0 MVP) 👓

Профессиональный чат-бот для салона оптики в Санкт-Петербурге. Автоматизирует запись на прием, информирование клиентов о готовности заказов и предоставляет удобную админ-панель.

## 🚀 Функционал

### Для клиентов:
- **Онлайн-запись:** Двухэтапный выбор (Дата -> Время) с проверкой занятых слотов в реальном времени.
- **Индикация занятости:** Занятое время помечается, нельзя записаться на прошедшее время.
- **Мои заказы:** Проверка статуса изготовления очков/линз по номеру заказа. Автоматические уведомления о готовности.
- **Личный кабинет:** Привязка номера телефона для связи заказов и уведомлений.
- **Авто-напоминания:** Фоновый процесс (APScheduler) отправляет уведомление клиенту за 2 часа до визита.
- **Информация о салоне:** Адрес и часы работы.
- **Помощь:** Подробные инструкции по использованию бота.

### Для администратора:
- **Админ-панель:** Закрытый раздел для управления салоном.
- **Список записей:** Просмотр всех предстоящих записей.
- **Все заказы:** Полный список заказов с их статусами и связанными клиентами.
- **Список клиентов:** Просмотр списка всех пользователей бота.
- **Полная карточка клиента:** Команда `Инфо [ID/Телефон]` для получения детальной информации о клиенте (профиль, визиты, заказы).
- **Управление статусом заказа:** Команда `Заказ [номер] [статус] [телефон_клиента (необязательно)]` для обновления статуса заказа.

## 🛠 Стек технологий
- **Язык:** Python 3.x
- **API:** [vk_api](https://github.com/python-vk-api/vk-api) (Long Poll)
- **База данных:** SQLite (файл `optica.db` для хранения записей, заказов, пользователей)
- **Планировщик:** [APScheduler](https://apscheduler.readthedocs.io/en/master/) (фоновые задачи для напоминаний)
- **Сборка для Windows:** [PyInstaller](https://pyinstaller.org/)
- **Интерфейс настройки:** Tkinter
- **OS:** Разработано на Linux, с автоматизированной сборкой для **Windows**.

## 📋 Инструкция по установке и запуску (для Администратора)

Для нетехнических пользователей достаточно скачать готовый файл!

1.  **Скачайте бота:**
    *   Перейдите в раздел [**Releases**](https://github.com/ВАШ_ПОЛЬЗОВАТЕЛЬ/ВАШ_РЕПОЗИТОРИЙ/releases) на GitHub (замените `ВАШ_ПОЛЬЗОВАТЕЛЬ/ВАШ_РЕПОЗИТОРИЙ` на свои данные).
    *   Найдите последний релиз (например, `v1.0.0`).
    *   В разделе "Assets" скачайте файл **`optica_bot.exe`**.

2.  **Первый запуск и настройка (один раз):**
    *   Дважды кликните на скачанный файл `optica_bot.exe`.
    *   Появится небольшое окно "Настройка Оптика-Бот".
    *   Введите ваш **Токен группы ВКонтакте** (или личного аккаунта, если используете его), ваш **цифровой ID ВКонтакте** (чтобы получить доступ к админ-панели) и **Название салона**.
    *   Нажмите "Завершить установку". Бот создаст файл `config.py` и `optica.db` (базу данных) рядом с `optica_bot.exe` и начнет свою работу в фоновом режиме. Консольное окно не появится.

3.  **Автозапуск бота при включении компьютера (Windows):**
    *   Нажмите `Win + R` на клавиатуре.
    *   В появившемся окне введите `shell:startup` и нажмите Enter. Откроется папка "Автозагрузка".
    *   Скопируйте **ярлык** (НЕ сам файл) на `optica_bot.exe` в эту папку.
    *   Теперь бот будет запускаться автоматически при каждом включении Windows.

4.  **Обновление бота:**
    *   Скачайте новую версию `optica_bot.exe` с GitHub Releases.
    *   Замените старый файл `optica_bot.exe` на новый, **но НЕ удаляйте и НЕ перезаписывайте файлы `config.py` и `optica.db`!** Они содержат ваши настройки и данные.
    *   Перезапустите компьютер или вручную завершите процесс старого бота и запустите новый.
