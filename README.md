# VK Booking Bot для салона оптики

Коммерческий проект: автоматизация записи клиентов и управления заказами через VK API.

## 🚀 Технологический стек
* **Язык:** Python 3.12
* **Библиотека:** `vk_api` (LongPoll)
* **База данных:** SQLite (хранение записей, клиентов и статусов заказов)
* **Инфраструктура:** Docker, Docker Compose
* **CI/CD:** GitHub Actions (автоматический деплой на удаленный Ubuntu-сервер через SSH)

## 📋 Основной функционал

### Для клиентов:
* **Запись на прием:** Интерактивный календарь с возможностью выбора даты на 30 дней вперед.
* **Личный кабинет:** Просмотр активных записей и истории обращений.
* **Мои заказы:** Отслеживание статуса текущих заказов (изготовление очков/линз).
* **Информационный блок:** Раздел "О салоне" и система помощи.

### Для администрации:
* **Админ-панель:** Встроенный интерфейс внутри бота для управления бизнесом.
* **Управление записями:** Просмотр всех актуальных записей на прием.
* **База клиентов:** Быстрый поиск и просмотр информации о клиентах.
* **Создание заказов:** Оперативное добавление новых заказов в базу данных.
* **Поддержка:** Специализированный раздел помощи для персонала.

## 🛠 Архитектура и безопасность
* **Контейнеризация:** Весь проект обернут в Docker, что гарантирует идентичность окружения в разработке и на продакшене.
* **Безопасность:** Конфиденциальные данные (токены, ID админов) вынесены в переменные окружения и управляются через GitHub Secrets.
* **Persistent Data:** База данных SQLite вынесена в Docker Volume для сохранения данных при пересборке контейнера.
* **Автоматизация:** Настроен CI/CD Workflow: при пуше в `master` бот автоматически проходит сборку и перезапускается на сервере.
