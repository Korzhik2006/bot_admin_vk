# VK Bot: Optica Express (v1.0 MVP) 👓

Профессиональный чат-бот для салона оптики в Санкт-Петербурге. Автоматизирует запись на прием и информирование клиентов о готовности заказов.

## 🚀 Функционал
- **Умная запись:** Двухэтапный выбор (Дата -> Время) с проверкой занятых слотов в реальном времени.
- **Индикация занятости:** Свободное время подсвечивается синим, занятое — белым и становится неактивным.
- **Статус заказов:** Мгновенная проверка готовности очков по номеру заказа из БД SQLite.
- **Авто-напоминания:** Фоновый процесс (APScheduler) отправляет уведомление клиенту за 2 часа до визита.
- **Админ-панель:** Закрытый раздел для сотрудников (просмотр всех записей, управление базой заказов).

## 🛠 Стек технологий
- **Язык:** Python 3.12
- **API:** [vk_api](https://github.com) (Long Poll)
- **База данных:** SQLite (хранение записей, заказов и состояний пользователей)
- **Планировщик:** APScheduler (фоновые задачи)
- **OS:** Разработано на Ubuntu Linux, готово к деплою на Windows.
